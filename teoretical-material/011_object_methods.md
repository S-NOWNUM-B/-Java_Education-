# Методы класса Object

Корень всей иерархии классов. Любой класс наследуется от `Object` неявно.

### Основные методы
*   `toString()`: Строковое представление объекта. По умолчанию: `ИмяКласса@ХэшКод`.
*   `equals(Object obj)`: Проверка на равенство. По умолчанию сравнивает ссылки (`==`).
*   `hashCode()`: Возвращает числовой код объекта.
*   `getClass()`: Возвращает метаданные класса во время выполнения (для рефлексии).
*   `clone()`: Создает копию объекта (требует интерфейс `Cloneable`).

### Контракт equals и hashCode
1.  Если `equals` возвращает `true`, то `hashCode` **обязательно** должны быть одинаковыми.
2.  Если `hashCode` одинаковые, это **не гарантирует**, что `equals` вернет `true` (коллизия).
3.  Если переопределяешь один, переопределяй и второй.
