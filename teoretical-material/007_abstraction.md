# Абстрактные классы

### Особенности
*   Ключевое слово: `abstract`.
*   Нельзя создавать объекты (`new AbstractClass()`).
*   Может содержать как обычные методы, так и абстрактные (без тела).

### Назначение
*   Задать общую структуру для группы связанных классов.
*   Реализовать общую логику, оставив специфику дочерним классам.

### Отличие от интерфейса
*   Класс может иметь состояние (обычные поля).
*   Класс может иметь конструкторы.
*   Ограничение на одиночное наследование.
