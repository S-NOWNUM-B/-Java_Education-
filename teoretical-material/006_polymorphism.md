# Полиморфизм

### Статический (Перегрузка / Overload)
*   Методы с одним именем, но **разными** параметрами.
*   Определяется на этапе компиляции.

### Динамический (Переопределение / Override)
*   Изменение логики метода родителя в дочернем классе.
*   Определяется во время выполнения (Runtime).
*   Аннотация `@Override` обязательна для контроля компилятором.

### Ссылочные переменные
`Parent obj = new Child();` — переменная типа родителя может ссылаться на объект потомка. Вызовется метод `Child`.
