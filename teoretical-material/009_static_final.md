# static и final

### Модификатор final
*   **Для переменной**: Константа (нельзя изменить значение после инициализации).
*   **Для метода**: Нельзя переопределить (Override) в потомках.
*   **Для класса**: Нельзя наследоваться от него (например, `String`).

### Модификатор static
*   **Поля**: Общее значение для всех объектов класса. Хранится в Metaspace.
*   **Методы**: Принадлежат классу, а не объекту. Можно вызывать без создания экземпляра.
*   **Блоки**: Выполняются один раз при загрузке класса в память.

### Ограничение static
Статические методы не могут обращаться к нестатическим полям (потому что их может не существовать в момент вызова).
