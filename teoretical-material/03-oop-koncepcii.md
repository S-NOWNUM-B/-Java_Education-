# Объектно-Ориентированное Программирование (ООП)

ООП — это парадигма программирования, основанная на концепции "объектов".

## Четыре столпа ООП

1. **Инкапсуляция**: Скрытие внутренней реализации и предоставление публичного интерфейса.
2. **Наследование**: Механизм, позволяющий одному классу наследовать поля и методы другого.
3. **Полиморфизм**: Возможность использовать один и тот же интерфейс для различных типов данных.
4. **Абстракция**: Выделение значимых характеристик объекта и игнорирование несущественных деталей.

## Классы и Объекты

- **Класс** — это "чертеж" или шаблон для создания объектов.
- **Объект** — это экземпляр класса.

```java
class Dog {
    String name;
    
    void bark() {
        System.out.println("Гав-гав!");
    }
}

// Создание объекта
Dog myDog = new Dog();
myDog.bark();
```
