# Интерфейсы

### Основы
*   Ключевое слово: `interface`.
*   Все поля по умолчанию: `public static final`.
*   Все методы без тела по умолчанию: `public abstract`.

### Новое (с Java 8/9)
*   `default methods`: Позволяют добавлять логику в интерфейс без поломки старых реализаций.
*   `static methods`: Утилитарные методы внутри интерфейса.
*   `private methods`: Для переиспользования кода внутри интерфейса.

### Маркерные интерфейсы
Интерфейсы без методов (напр. `Serializable`, `Cloneable`). Служат пометкой для JVM.
