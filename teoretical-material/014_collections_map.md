# Карты (Map)

Хранение данных в формате "Ключ — Значение". Ключи уникальны.

### HashMap
*   **Основа**: Массив корзин (Buckets). Внутри корзины — связный список или дерево (с Java 8).
*   **Порядок**: Не гарантируется.
*   **Хэширование**: Позиция объекта зависит от `hashCode()`.
*   **Коллизия**: Когда два разных ключа имеют один хэш и попадают в одну корзину.

### LinkedHashMap
*   **Особенности**: Сохраняет порядок добавления (insertion-order) или доступа (access-order).

### TreeMap
*   **Особенности**: Сортирует по ключам. Требует, чтобы ключи реализовывали `Comparable` или передавался `Comparator`.

### Hashtable
*   Устаревший аналог `HashMap`.
*   Методы синхронизированы (медленный в многопоточности).
*   Не позволяет `null` в ключах и значениях.
