# Generics (Дженерики)

Механизм для типизации коллекций и методов. Позволяет избежать `ClassCastException`.

### Wildcards
*   `<?>`: Любой тип.
*   `<? extends T>`: Ограничение сверху (Т или его потомки). Только чтение.
*   `<? super T>`: Ограничение снизу (Т или его предки). Только запись.

### Type Erasure (Стирание типов)
В скомпилированном байт-коде информации о дженериках нет — они заменяются на `Object` (или границу). Это нужно для обратной совместимости.

### Ограничения
1. Нельзя использовать примитивы (`List<int>` — нельзя, `List<Integer>` — можно).
2. Нельзя создать массив дженерика (`new T[10]`).
3. Нельзя использовать `instanceof T`.
