# Введение в Java

Java — это объектно-ориентированный язык программирования, разработанный компанией Sun Microsystems (в настоящее время принадлежит Oracle).

## Основные особенности Java

1. **Платформенная независимость**: Код компилируется в байт-код, который выполняется на JVM (Java Virtual Machine).
2. **Объектно-ориентированный подход**: Все в Java является объектом (за исключением примитивных типов).
3. **Строгая типизация**: Каждая переменная и выражение имеют тип, известный во время компиляции.
4. **Управление памятью**: Автоматическая сборка мусора (Garbage Collection).

## История Java

Java была выпущена в 1995 году. Её девиз: "Write Once, Run Anywhere" (WORA).

## JDK, JRE и JVM

- **JVM (Java Virtual Machine)**: Среда, исполняющая байт-код.
- **JRE (Java Runtime Environment)**: JVM + библиотеки для запуска приложений.
- **JDK (Java Development Kit)**: JRE + инструменты для разработки (компилятор javac и др.).
