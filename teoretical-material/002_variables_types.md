# Переменные и типы данных

### Примитивные типы (8 штук)
*   **Целые**: `byte` (1б), `short` (2б), `int` (4б), `long` (8б).
*   **Дробные**: `float` (4б), `double` (8б).
*   **Символ**: `char` (2б, UTF-16).
*   **Логика**: `boolean` (true/false).

### Ссылочные типы
*   Хранят адрес объекта в куче (Heap).
*   Дефолтное значение: `null`.
*   Примеры: `String`, `Массивы`, `Классы`.

### Stack vs Heap
*   **Stack**: Хранит примитивы и ссылки на объекты. Быстрый, очищается при выходе из метода.
*   **Heap**: Хранит сами объекты. Медленнее, очищается Garbage Collector-ом.
