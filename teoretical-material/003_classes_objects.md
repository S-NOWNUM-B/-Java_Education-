# Классы и Объекты

### Класс (Шаблон)
*   **Состав**: Поля (состояние) и Методы (поведение).
*   **Конструктор**: Специальный метод без возвращаемого типа для создания объекта. Имя совпадает с классом.

### Объект (Экземпляр)
*   Конкретная реализация класса в памяти.
*   Создается через `new`.

### Ключевое слово this
*   Ссылка на текущий экземпляр класса.
*   Используется для обращения к полям, если их имена совпадают с аргументами метода.

### Инициализация
1.  Статические блоки (один раз при загрузке класса).
2.  Нестатические блоки (при каждом создании объекта).
3.  Конструктор.
