## О проекте

> **Практический подход к изучению Java через реализацию реальных проектов**

Этот репозиторий содержит набор проектов различной сложности для освоения языка программирования Java. От простых консольных приложений до сложных систем с применением продвинутых концепций ООП и паттернов проектирования.
