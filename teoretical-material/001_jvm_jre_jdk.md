# JVM, JRE, JDK

### JVM (Java Virtual Machine)
*   Среда выполнения байт-кода.
*   Обеспечивает кроссплатформенность.
*   Управляет памятью (GC) и потоками.

### JRE (Java Runtime Environment)
*   **Состав**: JVM + Стандартные библиотеки (`rt.jar`).
*   **Назначение**: Только для запуска готовых программ.

### JDK (Java Development Kit)
*   **Состав**: JRE + Инструменты разработчика (`javac`, `jdb`, `jar`).
*   **Назначение**: Для написания и компиляции программ.

### Жизненный цикл кода
1.  `.java` (Исходник)
2.  `javac` (Компилятор)
3.  `.class` (Байт-код)
4.  `JVM` (Выполнение)
